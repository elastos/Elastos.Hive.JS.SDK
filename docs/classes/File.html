Class File

Internal class mimicing Java File class in order to reduce the divergence with Java implementation for now. NOTE: We could think about a totally different way to store items, and we will also need an abstraction layer to use different storages. But for now, we try to remain as close as the java implementation as we can until this SDK is totally stable.
